About Our Lab

We are a highly motivated, undergraduate student-led research lab at Morgan State University, dedicated to pushing the boundaries of technology in the fields of: Artificial Intelligence, Human-Robot Interaction, and Quantum Computing.

The lab operates under the guidance of Dr. Paul Wang, Chair and Professor of the Department of Computer Science. Our work emphasizes practical application, collaborative development, and cutting-edge software solutions to solve real-world challenges in STEM.

Project Summary

Under the supervision of Dr. Paul Wang, Chair of the Computer Science Department at Morgan State University, this project integrates OpenAI's GPT and Whisper models with the NAO6 humanoid robot to enhance its conversational and interactive capabilities. The system enables NAO to understand spoken language, generate intelligent responses, and interact naturally with users.

Key Achievements

  • Developed a Flask-based backend server for handling speech transcription, OpenAI responses, and database logging.
  • Implemented face recognition for personalized greetings and user identification.
  • Designed multi-mode personalities (e.g., study mode, general mode) with adjustable tone and pitch.
  • Currently expanding to Pinecone vector search for memory-based conversation and knowledge retrieval.

MSU CS Navigator (RAG Chatbot)

Project by Sakina Shrestha

Project Demo

Click image above to view GitHub

Project Summary

Built an AI-powered chatbot using **Retrieval-Augmented Generation (RAG)** to provide accurate, context-aware answers from a custom knowledge base. Integrated OpenAI, Pinecone, ViteReact, and deployed using FastAPI and Docker on AWS.

Key Achievements

  • Implemented GPT + Pinecone RAG architecture for dynamic document retrieval and response generation.
  • Deployed backend using FastAPI, Docker Compose, and AWS EC2.
  • Supported multi-platform access via web, Android app, and NAO humanoid robot.
  • Designed secure MySQL-based user authentication and for more security used JWTs.
  • Presented work at multiple research conferences and symposiums.
